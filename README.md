📱 About GameViewApp
GameViewApp is a mobile application developed as part of the FIAP curriculum. The primary goal of this project was to implement robust offline functionality within a Kotlin environment, ensuring that users can access and visualize critical information regardless of their internet connectivity.

🔑 Key Features
Persistent Caching: Efficient local storage implementation to keep data accessible offline.

Data Visualization: A clean UI built to display cached information seamlessly.

Modern Android Stack: Developed 100% in Kotlin following best practices for performance and scalability.

🛠️ Tech Stack
Language: Kotlin

Framework: Android SDK

Data Management: Local Caching (Room/SQL or SharedPreferences)

Architecture: (e.g., MVVM / Clean Architecture)
